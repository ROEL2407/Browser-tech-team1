• Progressive enhancement

    Focuses on the content of a webpage. Within your content, HTML is the most important. This is also the part that always needs to be displayed. After HTML, you can look for add-ons to make the page more usable and pleasurable. You can do this with CSS and JavaScript. Progressive enhancement can be divided into 4 steps:

    1. Functional

    The task needs to be executed no matter what.

    A boat needs to be able to float in the water no matter what, otherwise it wouldn't be a functional boat. The same thing is applied to a search bar, which consists of an action attribute on the form tag. It functions using only HTML.

  • 2. Reliable

    Always needs to work and in the same way

    People recognize/are familiar with search bars just like they are with boats. They know when they see a magnifying glass with a field to type in that it's a search bar.

  • 3. Usable

    Needs to be user-friendly

    With the use of javascript you can make the experience more user-friendly by having a placeholder to let the user know what kind of input they can give.

  • 4. Pleasurable

    Enjoyable to use

    A fancy yacht is more enjoyable than just a tiny wooden boat. By adding suggestions for the user while their typing, autocompleting their search input, some confetti shower and/or animations a person will enjoy it more than when you give no feedback at all.
